<h2>3)Omów typy zmiennych w Javascript</br></h2>
W JavaScript nie ma możliwości definiowania różnych typów zmiennych tak </br>
jak w C++, typy zmiennych są definiowane w trakcie ich przypisywania.</br>
Podstawowe zmienne liczbowe w JavaScript to:</br>
              Number → Typ liczbowy</br>
                       • zmienna całkowite</br>
                       • zmiennoprzecinkowe</br>
              String → Typ tekstowy</br>
              Boolea → Typ logiczny</br>
              Null → Pusta zawartość</br>
              Undefined → Typ niezidentyfikowany</br>
              Object → Obiekt</br>


<h2>4) Przy nadawaniu nazw zmiennym należy przestrzegać pewnych zasad:</br></h2>
(trzy zasady)</br>
</br>
Przy nadawaniu nazw zmiennym należy przestrzegać pewnych zasad:</br> 1. Nazwa zmiennej musi się zaczynać od litery, znaku "$" lub "_", może zawierać tylko litery (bez polskich znaków), cyfry oraz "$" i "_".</br>
<u>Poprawne nazwy zmiennych</u>: zmienna1, $wynik, _rozmiar, wynik1, Rekord, $_gracz</br>
<u>Błędne nazwy zmiennych</u>: 1zmienna, wynik gry, wysokosc&szerokosc, wynik.rekord, &gracz, wysokość, imie+nazwisko.</br>
2. Wielkość liter ma znaczenie.</br>
<u>Rekord i rekord </u>to dwie różne zmienne! Jest to bardzo częsta przyczyna błędów w kodzie początkujących programistów.</br>
3. Należy nie używać słów kluczowych JavaScript jako nazw zmiennych.</br>

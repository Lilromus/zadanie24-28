<h2>1. W jaki sposób można określić właściwość danego obiektu ( dwa sposoby wraz z
przykładami)? (odpowiedź poniżej)</h2>


<p class="ex">
<font size = "4" face="Arial"><u>Właściwość danego obiektu można określić (wpisać daną do pola
odpowiedzialnego za właściwość)</u></font></br>
<strong>Sposób 1</strong></br>
Używając zapisu postaci:</br>
a)</br>
<strong>nazwa_obiektu.nazwa_właściwości="tekst";</strong> -> dla właściwości typu tekstowego</br>
np.</br>
auto.marka="Opel Omega";</br>

np.</br>
auto.rok=1996;</br>
auto.cena=25000;</br>
<strong>Sposób 2</br></strong>
używając zapisu postaci:</br>
<strong>nazwa_obiektu[nazwa_właściwości];</br></strong>
np. auto[cena]=25000;</br>

</br>

np. console.log(samochod1.kolor); → wyświetla kolor obiektu samochod</br>

function klient(nazwisko_k,imie_k,zawod_k)</br>
{</br>
this.nazwisko=nazwisko_k;</br>
this.imie=imie_k;</br>
this.zawod=zawod_k;</br>
</br>
wypisz = function ()</br>
{</br>
alert(this.nazwisko + ' ' + this.imie)</br>
}</br>
}</br>

</p>

<h2>6. Konstruktor w JS na podstawie przykładu wraz z opisem.</h2>
<font size = "4" face="Arial"></font>Chcemy utworzyć ich kilka. Każdy obiekt powinien posiadać jakieś właściwości i metody np. width,</br>
height i wypisz().</br>
Aby utworzyć kilka podobnych obiektów posłużymy się klasą obiektu.</br>
W JS w przeciwieństwie do innych języków nie mamy mechanizmu konstruktora, ale samą klasę</br>
możemy stworzyć za pomocą zwykłej funkcji:</br>
</font>
<p class="ex">
function SuperObjectClass(_width,_height) {</br>
this.width = _width;</br>
this.height = _height;</br>
this.print = function() {</br>
console.log(this.width + 'x' + this.height)</br>
}</br>
}</br>
var myObject1 = new SuperObjectClass(200, 100);</br>
var myObject2 = new SuperObjectClass(300, 200);</br>
myObject1.print(); //wypisze 200x100</br>
myObject2.width = 600;</br>
myObject2.print() //wypisze 600x200</br>
</p>
<h2>7. Znaczenie słowa kluczowego<strong> New</strong> wraz z przykładem.</h2>

<p class="ex">

Do utworzenia <strong><u>nowego</u></strong> obiektu na podstawie konstruktora stosowane jest <strong>słowo kluczowe new</strong>.

<h2>8. Znaczenie słowa kluczowego prototype wraz z przykładem i opisem ( szczególnie ważny
tekst pogrubiony).</h2>
<p class="ex">
<font size = "4" face="Arial"><u>Właściwość prototype
</u></font></br>
Do definowania metod i właściwości dla obiektu jest wykorzystana właściwości <strong>prototype</strong>.</br>
</br>
Przykład</br>
</br>
function klient()</br>
{</br>
this.nazwisko='Bielski';</br>
this.imie='Paweł';</br>
}</br>
klient.prototype.pisz_dane = function ()</br>
{</br>
document.write(this.nazwisko + ' ' + this.imie)</br>
}</br>
klient.prototype.zawod = 'kierowca';</br>
var osoba1 = new klient();</br>
osoba1.pisz_dane();</br>




Opis do przykładu powyżej:</br>
</br>
W definicji konstruktora nie zostały zadeklarowane żadne metody i właściwości. Dopiero po użyciu</br>
właściwości prototype została dodana metoda pisz_dane oraz właściwość zawod. Od tej pory każdy</br>
nowo tworzony obiekt na podstawie konstruktora klient będzie posiadał tę dodatkową właściwość i</br>
metodę.</br>
</br>
<strong>Właściwość prototype może być również wykorzystana do dodawania dodatkowych</br>
metod lub właściwości do istniejących obiektów.</br></strong>
</p>
